# Crimson Slash

A modern full-stack web application built with Ruby on Rails and React, using Inertia.js for seamless server-side rendering with a React frontend.

## Tech Stack

### Backend
- **Ruby on Rails** 8.1.1
- **PostgreSQL** database
- **Inertia Rails** for SPA-like experience without API complexity

### Frontend
- **React** 19 with **TypeScript**
- **Inertia.js** for seamless Rails-React integration
- **Tailwind CSS** 4 for styling
- **Radix UI** & **Headless UI** for accessible components
- **Vite** for fast asset bundling

### Development & Deployment
- **Kamal** for deployment
- **Docker** support
- **Vite Rails** for asset pipeline
